# Šablona posterů UPa (neoficiální).

## Proč použít tuto šablonu?

Šablona obsahuje loga a barvy všech fakult. Pro změnu stačí nahradit zkratku fakulty v příkazu **\newcommand{\faculty}{_DFJP_}** zkratkou jiné (z výběru: _DFJP_, _FEI_, _FCHT_, _FES_, _FR_, _FZS_, _FF_ a _UPA_).

## Jak začít?

Nainstalujte si [TexStudio](https://www.texstudio.org/), stáhněte a otevřete soubory zde uložené a můžete začít editovat. V _TexStudiu_ lze z kódu snadno zkompilovat _.pdf_ soubor.

## Poděkování

Šablona je založena na originále od P. Dreuwa a T. Deselaerse, který je dostupný na https://github.com/deselaers/latex-beamerposter a poděkování za spolupráci patří rovněž P. Filipovi, u nějž najdete jinou verzi šablony UPa využitelnou jak pro postery, tak pro prezentace: https://github.com/petrfilip/upce-beamer-template
